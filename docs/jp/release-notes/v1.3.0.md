# リリースノート: `v1.3.0`

`wg-studio v1.3.0` は、運用者が「今どうなっているか」を読み取りやすくするための UX / visibility 強化リリースです。

## 主な変更

- Dashboard の右上を `runtime / system` 情報パネルへ刷新し、`runtime status`、`uptime`、`container`、最終同期系の情報を表示
- Dashboard の主要パネルを slot ベースに整理し、今後の差し替えや user 別配置に備えた構造へ移行
- トラフィック推移を期間切替、縦横軸、凡例、拡大表示つきの読みやすいグラフへ改善
- `Group -> User -> Peer` を直接返す topology API を追加し、将来の 2D / 3D 構成図の土台を準備
- トラフィックスナップショットに保持日数設定を追加し、古い履歴を自動削除するように変更

## ねらい

この版の目的は、単に画面を整えることではなく、平常時でも異常時でも「運用判断に必要な情報がすぐ見える」状態に寄せることです。

とくに `runtime drift` が起きていない通常時でも、`uptime` や `runtime status`、トポロジの全体像、履歴グラフの変化が読み取れるようにしています。

## 運用メモ

- `設定 > Language and time` に `トラフィック履歴保持日数` を追加しました
- 保持期間を超えた `peer_traffic_snapshots` は、snapshot 取得時に自動削除されます
- topology API は将来のグラフ表示向けの土台であり、この版では Dashboard の軽量プレビューとして利用しています
