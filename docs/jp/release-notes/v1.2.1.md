# リリースノート: `v1.2.1`

`wg-studio v1.2.1` は、`1.2.0` の frontend seam 整理に対する薄い polish 版です。

## 主な内容

- Settings の interface 保存ボタンと error log level に安定した selector を追加
- Playwright smoke で Settings の interface MTU 保存を確認するようにした
- Playwright smoke で GUI の error log level 保存と Logs 画面での反映確認を追加した
- 変更範囲は frontend の信頼性補強と、operator が触る Settings の見える挙動に限定した

## この版の意味

`1.2.1` は frontend 分離テーマを広げる版ではありません。

`1.2.0` で Settings が持つ責務が少し増えたので、特に次の値まわりの確認を厚くした版です。

- interface MTU
- GUI error log level

## 運用メモ

- 意図的な運用手順の再設計はありません
- `1.2.0` よりも、Settings 保存まわりが browser smoke で直接確認されます
