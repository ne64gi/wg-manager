# リリースノート v1.1.5

## 概要

`v1.1.5` は、起動直後の operator entrypoint を扱いやすくし、GUI 側にも小さな runtime 可視化を追加した版です。

## 変更点

- `scripts/stack.sh` と `scripts/stack.ps1` に `wait` コマンドを追加
- `health` は API の readiness を待ってから web 到達性まで確認するように改善
- `smoke` は readiness preflight を通してから Playwright を実行するように改善
- Playwright smoke は、より安定した UI root 待機と正規化済みネットワーク値を使うように改善
- Settings の `ビルド情報` に現在の runtime adapter 表示を追加

## 意味

- `up` の直後に wrapper コマンドを叩いても、起動待ち不足でこけにくくなります
- 「API の準備不足」と「smoke 自体の失敗」を切り分けやすくなります
- 現在どの runtime adapter で動いているかを GUI から確認できます
