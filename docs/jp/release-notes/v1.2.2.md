# リリースノート: `v1.2.2`

`wg-studio v1.2.2` は、ランタイムの設定ドリフトと互換性にフォーカスした安定化リリースです。

## 変更点

- インターフェース再生成後の `wg0` 設定ドリフトを修正（AllowedIPs / ルート状態の再適用）
- 初期設定で `server_address` / `server_dns` 変更時に自動再適用を追加
- 初期設定 API/UI に `server_address` / `server_dns` の永続化フィールドを追加
- Docker API バージョンのハードコードを除去。設定 `docker_api_version` を使用して対応バージョンを制御

## 背景

このリリースで、`wg0` が再作成された際に必要なルートやピア構成が失われてしまう不具合を解決し、Docker バージョン差異による起動失敗をなくしました。

## 運用メモ

- 設定画面でサーバーアドレスとDNSを確認・更新してください
- Docker API バージョン設定は再起動不要で反映される設計です
- 1.2.1 のUIポリッシュをベースにした安定化リリースです
