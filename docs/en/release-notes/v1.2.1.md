# Release Notes: `v1.2.1`

`wg-studio v1.2.1` is a thin frontend-polish follow-up to the `1.2.0` seam work.

## Highlights

- added stable Settings selectors for interface-save and error-log-level controls
- extended Playwright smoke so it verifies interface MTU persistence through Settings
- extended Playwright smoke so it verifies GUI error-log-level persistence through Settings and Logs
- kept the change narrowly scoped to frontend confidence and operator-facing Settings behavior

## Why It Matters

`1.2.1` does not widen the frontend-separation theme.

It tightens confidence around the Settings screen that now carries more operator-visible state, especially:

- interface MTU
- GUI error-log threshold

## Operator Notes

- no intended workflow redesign in this release
- Settings persistence is covered a little more directly by browser smoke than in `1.2.0`
